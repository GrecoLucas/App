# SmartShop - Lista de Compras Inteligente
## Sobre o Projeto
O SmartShop é um aplicativo móvel desenvolvido em Flutter que revoluciona a experiência de fazer compras, oferecendo uma solução completa para gerenciamento de listas de compras com recursos avançados como scanner de código de barras, controle de orçamento, compartilhamento de listas em tempo real e funcionamento offline/online.
